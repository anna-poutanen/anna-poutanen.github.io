# Anna Poutanen's Portfolio Website

A modern, interactive portfolio website showcasing my projects, experience, and skills as a Software Engineering student at the University of Waterloo.

## Live Site

Visit the live site at: [anna-poutanen.github.io](https://anna-poutanen.github.io)

## Features

- **Modern Design**: Dark theme with glassmorphism effects and gradient accents
- **Fully Responsive**: Optimized for desktop, tablet, and mobile devices
- **Interactive Animations**: 
  - Cursor glow effect
  - 3D card tilt on hover
  - Scroll-triggered reveals
  - Parallax effects
  - Smooth scrolling navigation
- **Sections**:
  - Hero section with animated text
  - About me with technical skills
  - Experience timeline
  - Featured projects in bento grid layout
  - Contact information

## Technologies Used

- **HTML5** - Structure and content
- **CSS3** - Styling with custom properties, flexbox, and grid
- **JavaScript** - Interactive features and animations
- **GitHub Pages** - Hosting

## Design Features

- Custom color palette with primary (#6366f1), secondary (#ec4899), and accent (#14b8a6) colors
- Glassmorphism UI elements with backdrop blur
- Asymmetric bento box grid layout for projects
- Timeline design for work experience
- Animated gradient backgrounds

## Getting Started

To run this project locally:

1. Clone the repository:
   ```bash
   git clone https://github.com/anna-poutanen/anna-poutanen.github.io.git
   ```

2. Navigate to the project directory:
   ```bash
   cd anna-poutanen.github.io
   ```

3. Open `index.html` in your browser or use a local server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js with http-server
   npx http-server
   ```

4. View the site at `http://localhost:8000`

## Customization

To customize this template for your own use:

1. **Update Content**: Edit `index.html` to change text, links, and information
2. **Change Colors**: Modify CSS variables in `:root` section of `style.css`
3. **Add Projects**: Add new project cards in the projects section
4. **Update Skills**: Modify the skills tags in the about section
5. **Change Animations**: Adjust timing and effects in `script.js`
